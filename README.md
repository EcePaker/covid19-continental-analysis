# COVID-19 Continental Analysis

## Project Overview
This project analyzes COVID-19 data using SQL, Google Sheets, and Tableau.

## Tools Used
- SQL (DBeaver)
- Google Sheets
- Tableau

## Workflow
1. COVID-19 data queried using SQL
2. Query outputs imported into Google Sheets
3. Data cleaned and structured
4. Interactive dashboards built in Tableau

## Data Source
- Kaggle COVID-19 Dataset
